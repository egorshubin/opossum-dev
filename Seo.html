Google-Friendly FAQ Blocks With Schema Markup for SEO

Put question-answer items in any part of your special product page – or on all pages at once. FAQs can be added to any page you choose, bringing value to the customer. This FAQ module is Google-friendly and improves your SEO. It uses Google Schema markup. Your page’s rich snippet in Google search results will be larger in size, because it will contain your FAQs.

Types of content:

  • A separate FAQ page
  • Multiple FAQ lists

Use Hooks to position lists:

  • All the hooks available in your theme!
  • Create your own custom hooks (unlimited quantity!)
  • Arrange many lists in one hook (drag and drop positions)

Set your display rules for each list:

  • By language
  • By currency
  • By customer group
  • By page

Display by pages:

  • All pages
  • Product pages
  • Categories
  • CMS pages
  • Home page, Contact us, Best sales, Cart, and other pages

Use Shortcodes to insert right into the contents of:

  • CMS pages
  • Product description and summary
  • Category description (for Prestashop 1.6.1 and 1.7.1 and newer)

Add individual styles:

  • CSS classes
  • HTML tags
  • Accordion

Some more features:

  • Show/hide list’s title
  • Enable/disable Google Schema Markup
  • Duplicate lists
  • Multistore support
  • Add content in all of your languages
  • Mobile responsive interface
  • Use one FAQ item many times in different lists, and edit it in one place
  • Add HTML to question and answer fields. WYSIWYG or direct HTML inpu
  • Add images to FAQ items text
  • FAQ lists are cached in database for excellent performance

A separate FAQ page

There's one pre-created FAQ page. It can be disabled. If you need more FAQ pages, you can easily create CMS pages in the admin, and then insert in their contents your list of questions via shortcode.
